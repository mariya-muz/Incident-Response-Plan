# Cybersecurity Incident Response Plan – Internee.pk

Project focused on structured procedures for threat detection, ransomware response, recovery, and employee awareness training. Demonstrated practical workflows using industry-standard practices and attack simulations.

---

## Internship Deliverables

1. **Incident Response Plan**
    
    Structured lifecycle: Preparation, Identification, Containment, Eradication, Recovery, Lessons Learned.
    
    *Context: Incident Response Plan document.*
    
2. **Threat Detection**
    
    Methods: Antivirus, Firewall, Event Logs, Network Monitoring.
    
    Tools: Windows Defender, Wireshark, Event Viewer.

2. **Ransomware Simulation**
    
    Phishing-based infection → Malware execution → File encryption → Detection → Containment.
    
    *Context: Simulation report.*
    
3. **Attack Timeline**
    
    Chronological record: phishing email, execution, encryption, alerts, isolation.
    
    *Context: Timeline table in report.*
    
4. **Containment Procedures**
    
    Disconnect systems, block access, disable accounts, restrict drives.
    
    *Context: Containment workflow in report.*
    
5. **Recovery**
    
    Antivirus scans, file removal, patching, backup restoration, password resets.
    
    *Context: Recovery procedures in report*
    
6. **Staff Training**
    
    Phishing awareness, password hygiene, safe browsing, incident reporting.
    
    *Context: Awareness guide in Staff Training report.*
    
7. **Response Workflow**
    
    Escalation: Employee → IT → Security → Management.
    
    *Context: Reporting flow.*
